Team Page validation report

Issue: The page used an h4 heading immediately after an h2 heading. This skips the h3 heading level, creating an incorrect heading hierarchy. Why it is a problem: Headings should follow a logical order (h1 → h2 → h3 → h4). Skipping heading levels makes the page structure confusing for screen readers and reduces accessibility. How it was resolved: The h4 headings were changed to h3 (or an appropriate h3 heading was added before the h4). This created a proper heading hierarchy and removed the validation error.
